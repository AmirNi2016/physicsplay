\makeproblem{Spin-1 rotations.}{gradQuantum:problemSet6:3}{ 
%\makesubproblem{}{gradQuantum:problemSet6:3a}

Consider angular momentum \( j = 1 \).  Express \( \bra{j = 1, m'} \hat{J}_y \ket{ j = 1, m } \) as a \( 3 \times 3 \) matrix.  Show that for \( j = 1 \), we can replace (units where \( \Hbar = 1 \)),

\begin{dmath}\label{eqn:gradQuantumProblemSet6Problem3:20}
e^{-i \hat{J}_y \beta}
=
1 - i \hat{J}_y \sin\beta - \hat{J}_y^2 \lr{ 1 - \cos\beta }.
\end{dmath}
} % makeproblem
