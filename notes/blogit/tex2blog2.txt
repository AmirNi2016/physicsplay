Rough notes of what's required in a new tex2blog script

- title and prologue
- \\section -> <h2> ... </h3>
- \\paragraph or \\subsection -> <h3> ...
- $...$ -> \(...\)
- \cref{FigN} -> fig. N
- \cref -> \ref
- \begin{itemize} -> <ul>
- \end{itemize} -> </ul>
- \item -> <li>
- {\em ...} -> <em> ... </em>
